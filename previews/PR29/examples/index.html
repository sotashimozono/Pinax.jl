Examples

A small catalogue of galleries built with Pinax. Every example is a short, self-contained Pinax script — compute something, describe a manuscript, then render it into a gallery. Each example page below shows that script's source verbatim; open the compiled gallery beside it.

A Pinax script reads like this:

using Pinax

@page :results "Results" begin
    @section :energy "Energy" begin
        @desc md"Energy density $E/N$ versus inverse temperature."
        @figure plot_energy()
        @caption "χ-convergence"
    end
end

render(out = "gallery")
